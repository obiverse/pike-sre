# @obiverse/pike-sre

## 0.2.0

### Minor Changes

- 1179ad5: Initial release of Pike's Structural Regular Expressions for TypeScript.

  Features:

  - Core commands: x, y, g, v, p, d, c, s, n, l, pipe
  - Fluent API with sre() builder
  - Lexer/tokenizer utilities with createLexer()
  - Pattern engine for reactive document processing
  - Template substitution and glob matching
  - Universal runtime support (Browser, Node, Deno, Bun)
  - Tree-shakeable with multiple entry points
